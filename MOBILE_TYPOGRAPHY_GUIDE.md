# 📱 Mobile Typography Guide - Font Sizes

## 🎯 **Optimized Font Sizes for Mobile**

All font sizes are carefully chosen for optimal readability on mobile devices while preventing iOS auto-zoom.

---

## 📏 **Typography Scale**

### Base Settings:
```css
html { font-size: 16px; }  /* Prevents iOS zoom */
body { 
    font-size: 16px; 
    line-height: 1.6;
}
```

---

## 🔤 **Headings**

### H1 - Page Titles

- **Weight:** 700 (Bold)
- **Line Height:** 1.2
- **Use:** Main page titles, hero headings
- **Example:** "My Properties", "Dashboard"

### H2 - Section Titles
- **Size:** 28px
- **Weight:** 700 (Bold)
- **Line Height:** 1.3

## 📐 **Spacing Guidelines**

### Line Heights:
- **Tight:** 1.2 (Headings)
- **Normal:** 1.5 (UI elements)
- **Relaxed:** 1.6 (Body text)
- **Loose:** 2.0 (Special cases)

### Letter Spacing:
- **Tight:** -0.5px (Large headings)
- **Normal:** 0px (Default)
- **Wide:** 0.5px (Uppercase labels)

---

## 🎨 **Font Weights**

- **Light:** 300
- **Regular:** 400 (Default)
- **Medium:** 500 (Buttons, labels)
- **Semi-bold:** 600 (Headings, emphasis)
- **Bold:** 700 (Titles, stats)

---

## ✅ **Best Practices**

### DO:
- ✅ Use 16px minimum for body text
- ✅ Use 16px for form inputs (prevents iOS zoom)
- ✅ Use line-height 1.6 for readability
- ✅ Use proper font weights for hierarchy
- ✅ Test on real devices
- ✅ Consider accessibility (WCAG AA)

### DON'T:
- ❌ Use font sizes below 12px
- ❌ Use font sizes below 16px for inputs
- ❌ Use line-height below 1.2
- ❌ Use too many font sizes
- ❌ Forget about contrast ratios
- ❌ Ignore touch target sizes

---

## 🧪 **Testing Checklist**

- [ ] Text is readable without zooming
- [ ] No iOS auto-zoom on input focus
- [ ] Proper hierarchy is clear
- [ ] Touch targets are min 44px
- [ ] Contrast ratios meet WCAG AA
- [ ] Works in dark mode
- [ ] Looks good on small screens (320px)
- [ ] Looks good on large screens (768px)

---

## 📱 **Device-Specific Considerations**

### iPhone SE (320px width):
- Minimum font size: 14px
- Comfortable reading: 16px
- Headings: 24-28px

### iPhone 12/13 (390px width):
- Minimum font size: 14px
- Comfortable reading: 16px
- Headings: 28-32px

### iPad Mini (768px width):
- Minimum font size: 15px
- Comfortable reading: 17px
- Headings: 32-36px

## 🎯 **Quick Reference**

```css
/* Copy-paste ready */
@media (max-width: 768px) {
    body { font-size: 16px; line-height: 1.6; }
    h1 { font-size: 32px; font-weight: 700; }
    h2 { font-size: 28px; font-weight: 700; }
    h3 { font-size: 24px; font-weight: 600; }
    h4 { font-size: 20px; font-weight: 600; }
    h5 { font-size: 18px; font-weight: 600; }
    h6 { font-size: 16px; font-weight: 600; }
    p { font-size: 16px; line-height: 1.6; }
    small { font-size: 14px; }
    .btn { font-size: 15px; min-height: 44px; }
    .form-control { font-size: 16px; min-height: 48px; }
}
```
