# Landlord Guide: Flexible Rental Duration Configuration

## Overview
The EasyRent platform now supports **complete flexible rental durations** for apartments, allowing landlords to offer multiple rental options to tenants.

## ✅ Supported Rental Duration Types

Your apartments can now support the following rental types:

1. **Hourly** - For short-term rentals (e.g., event spaces)
2. **Daily** - For daily rentals (e.g., vacation rentals)
3. **Weekly** - For weekly rentals
4. **Monthly** - Traditional monthly rentals
5. **Quarterly** - 3-month rentals (auto-calculated from monthly rate)
6. **Semi-Annually** - 6-month rentals (auto-calculated from monthly rate)
7. **Yearly** - 12-month rentals
8. **Bi-Annually** - 24-month rentals (auto-calculated from monthly rate)

## 🏠 How to Configure Rental Durations When Adding Apartments

### Step 1: Create Your Property
1. Go to the **Listing** page
2. Fill in your property details
3. Click "Create Property"

### Step 2: Add Apartments with Rental Duration Types
After creating your property, you'll see the apartment creation form with these fields:

| Field | Description | Required |
|-------|-------------|----------|
| **Tenant ID** | Optional - leave blank for vacant apartments | No |
| **From** | Start date (if tenant assigned) | No |
| **To** | End date (if tenant assigned) | No |
| **Price** | The rental amount for the selected duration type | Yes |
| **Rental Type** | Select the duration type (Daily, Weekly, Monthly, etc.) | Yes |

### Step 3: Select the Appropriate Rental Type
Choose the rental type that matches your pricing:

- **Monthly**: If your price is ₦50,000 per month, select "Monthly"
- **Daily**: If your price is ₦5,000 per day, select "Daily"
- **Weekly**: If your price is ₦30,000 per week, select "Weekly"
- **Quarterly**: If your price is ₦150,000 for 3 months, select "Quarterly"
- **Semi-Annual**: If your price is ₦300,000 for 6 months, select "Semi-Annual"
- **Yearly**: If your price is ₦600,000 per year, select "Yearly"
- **Bi-Annual**: If your price is ₦1,200,000 for 24 months, select "Bi-Annual"

### Step 4: Add Multiple Apartments
Click the **"+ Add Apartment"** button to add more apartments to your property. Each apartment can have a different rental type and price.

## 📝 Editing Existing Apartments

### To Update Rental Duration Configuration:

1. Go to your **Dashboard**
2. Find the apartment you want to edit
3. Click **"Edit Apartment"**
4. Scroll to the **"Supported Rental Types"** section

### Configuring Multiple Rental Options:

You can enable multiple rental types for a single apartment:

1. **Check the rental types** you want to support (e.g., Daily, Weekly, Monthly)
2. **Enter the rate** for each selected type
3. **Auto-calculated rates**: Quarterly, Semi-Annual, and Bi-Annual are automatically calculated from your monthly rate
4. **Set a default rental type** - This is what tenants will see first

### Example Configuration:

**Apartment: 2-Bedroom Flat**
- ✅ Daily Rental: ₦5,000 per day
- ✅ Weekly Rental: ₦30,000 per week  
- ✅ Monthly Rental: ₦100,000 per month
- ✅ Quarterly: ₦300,000 (auto-calculated: 3 × monthly)
- ✅ Semi-Annually: ₦600,000 (auto-calculated: 6 × monthly)
- ✅ Yearly: ₦1,200,000 per year
- **Default**: Monthly

## 💰 How Tenants Will See Your Rental Options

When tenants view your apartment through an invitation link or payment form, they will:

1. See all available rental duration types you've configured
2. Select their preferred duration type from a dropdown
3. Enter the quantity (e.g., 3 months, 2 weeks)
4. See the total amount calculated automatically
5. Complete payment securely

### Example Tenant Experience:

**Tenant selects:**
- Duration Type: Quarterly
- Quantity: 2 (meaning 2 quarters = 6 months)
- **Total Amount**: ₦600,000 (automatically calculated)

## 🔧 Technical Details

### Auto-Calculation Rules:

The system automatically calculates certain rental types from your monthly rate:

- **Quarterly** = Monthly Rate × 3
- **Semi-Annually** = Monthly Rate × 6
- **Bi-Annually** = Monthly Rate × 24

This ensures consistency and makes it easy to offer multiple options without manual calculation.

### Rate Conversion:

If you only set a monthly rate, the system can automatically offer:
- Quarterly rentals (3 months)
- Semi-annual rentals (6 months)
- Yearly rentals (12 months)
- Bi-annual rentals (24 months)

## 📊 Best Practices

### 1. Pricing Strategy:
- **Daily rates**: Typically 3-5% of monthly rate
- **Weekly rates**: Typically 20-25% of monthly rate
- **Yearly rates**: Often discounted (10-12 months' worth)

### 2. Target Market:
- **Hourly/Daily**: Event spaces, vacation rentals
- **Weekly**: Business travelers, short-term stays
- **Monthly**: Traditional tenants
- **Quarterly/Semi-Annual**: Corporate housing, students
- **Yearly/Bi-Annual**: Long-term tenants (with discounts)

### 3. Vacancy Management:
- Enable **multiple rental types** to attract different tenant segments
- Use **daily/weekly** options during low-demand periods
- Offer **yearly** options with discounts for stable income
