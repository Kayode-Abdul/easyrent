🚀 No Flash Test

This page tests the fix for the flash of light content when navigating between pages in dark mode.

Hero Section

Background should be visible without flash

Test Instructions:

  1. Enable dark mode using the toggle button (🌙 → ☀️ → 🕐)
  2. Navigate between different pages using the links above
  3. Observe if there's any flash of light content
  4. The page should load directly in dark mode without any flash

What Was Fixed:

  • Critical CSS: Inline styles in HTML head apply dark mode instantly
  • Immediate Script: JavaScript runs before DOM to set dark mode attribute
  • No Transitions: Disabled transitions on initial load to prevent flash
  • Background Fix: Hero section background properly handled in dark mode

Technical Details:

FOUC (Flash of Unstyled Content) was caused by:

  • Dark mode CSS loading after the page content
  • JavaScript applying dark mode after DOM ready
  • Transitions causing visible switching

Solution: Apply dark mode immediately in HTML head before any content loads.
